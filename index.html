Latte is a high-performance domain specific language for deep neural networks. It was designed to improve the programmability of neural networks by introducing an abstraction that reflects the concepts used by domain experts. Under the hood Latte leverages domain specific optimizations to emit efficient machine code to execute user-described networks.
