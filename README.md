# Latte.jl

## Quick Install
```
$ julia -e 'Pkg.clone("https://github.com/IntelLabs/Latte.jl")'
```

## MNIST
```
$ cd ~/.julia/v0.4/Latte/examples/mnist/data
$ ./get-data.sh
$ cd ..
$ julia mnist.jl
```

## cifar10
```
$ cd ~/.julia/v0.4/Latte/examples/cifar10/data
$ ./get-data.sh
$ cd ..
$ julia vgg-mini.jl
```
